1. Tổng Quan Sản Phẩm

VinhKhanhGuide là hệ thống hướng dẫn ẩm thực đa ngôn ngữ cho phố Vĩnh Khánh. App MAUI phục vụ khách tham quan bằng bản đồ, GPS/geofence, tour, QR, TTS/audio và lịch sử nghe. WebAdmin phục vụ Admin và chủ cửa hàng quản trị dữ liệu, duyệt POI, tạo tour, quản lý audio, QR, analytics và tài khoản. Backend API là nguồn dữ liệu dùng chung.

Nguyên tắc đối chiếu: WebAdmin đọc/ghi qua PoiApiClient, TourApiClient, AudioGuideApiClient, ListeningHistoryApiClient, ActiveDeviceApiClient, còn API lưu dữ liệu bằng repository JSON trong VKFoodAPI/App_Data. App MAUI đọc API trước, sau đó fallback sang SQLite/offline snapshot, cached snapshot hoặc seed data.
